## Requirements

- [Node.js](https://nodejs.org/)
- [Git](https://git-scm.org/)

## Setting up

Clone the [Template Project](https://github.com/hypermodeAI/template-project)

```bash
git clone https://github.com/hypermodeAI/template-project

cd template-project
```

Install dependencies in `./functions`

```bash
cd functions
```

<CodeGroup>
```bash npm
npm install
```

```bash yarn
yarn install
```

```bash bun
bun install
```
</CodeGroup>

Build project

```bash npm
npm run build
```
